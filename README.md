## About WorkPilot

WorkPilot is an institutional-grade Employee Management and Self-Service platform designed for modern enterprises. It provides a robust, high-fidelity experience for managing the entire employee lifecycle—from recruitment and onboarding to payroll, attendance, and offboarding.

### Core Features

- **Employee Self-Service (ESS):** User-friendly portal for record management, personal data updates, and leave requests.
- **Advanced Attendance & Biometrics:** Real-time tracking with digital horizontal gauges and automated payroll integration.
- **Recruitment & Applicant Tracking:** Streamlined HIRING process with pipeline management and resume tracking.
- **Institutional Analytics:** Data-driven reporting on workforce planning, performance, and engagement metrics.
- **Performance Management:** Automated appraisals and feedback collection.
- **Compliance & Document Management:** Centralized vault for HR documents and regulatory compliance records.

## Technical Foundation

- **Backend:** Laravel (PHP 8.2+)
- **Frontend:** Vite, Tailwind CSS 4, React
- **Real-time:** Laravel Reverb / Socket.io
- **Design System:** Anchored Midnight Navigation with Institutional Slate tokens.
